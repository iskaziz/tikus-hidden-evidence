# TIKUS: Hidden Evidence — Eye-Level Clean Prototype

This is a cleaned two-level prototype for the eye-level revamp.

Kept features:
- 25-second timer
- Score on top right
- Clue list footer
- Room name badge on bottom right
- Feisk logo on top left
- Start game message
- Level complete message
- Sound effects
- Clue-only editor mode
- Ambient lights and dust
- Final level report
- Random side character per level iteration

Removed:
- Old six-level structure
- Mouse feature
- Wrong-click feature
- Zone editor
- Hint systems / unused overlays / old mobile mode systems

New levels:
- Sitting Room
- Kitchen

- Enter code 0707
- Drag clues directly on the canvas
